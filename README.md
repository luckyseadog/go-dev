# Metrics Collection and Alerting Service

Tech Stack:
- Golang
- gRPC
- HTTP/HTTPS
- Chi
- Goroutines
- Encryption
- SQL

## How to run Server

```
cd cmd/server
go run .
```

## How to run Agent

```
cd cmd/agent
go run .
```

## How to run tests
```
go test ./...
```
